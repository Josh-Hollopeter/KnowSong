Spotify Web API

Know Song uses Spotify's public facing data retrieval system. User data is protected through OAuth2, a security protocol that ensures unaffiliated developer applications only have access to non sensitive information.

Do you know music? As a Know Song user, you have access to the entire spotify music library! Challenge yourself and try out some of our dynamically generated trivia games based on your favorite music artists!




Humble Beginnings

Know Song began as a final project for Skill Distillery's 16 week coding bootcamp in Denver, Colorado. Feel free to get in touch with the three developers Josh Hollopeter, George Moore, and Neal Savant.
